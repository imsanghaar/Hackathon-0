# Company Handbook

## Rules

1. **Always log important actions**
   - Record significant operations in System_Log.md

2. **Never take destructive actions without confirmation**
   - Always confirm before deleting, overwriting, or modifying critical data

3. **Move completed tasks to Done**
   - Once a task is finished, move it from Inbox/Needs_Action to the DOne folder

4. **Keep task files structured**
   - Maintain consistent formatting across all task files

5. **If unsure, ask for clarification**
   - When requirements are ambiguous, request clarification before proceeding
